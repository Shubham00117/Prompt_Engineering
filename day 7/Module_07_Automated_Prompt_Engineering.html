2Ô∏è‚É£ Prompt Search Algorithms

Core Idea: Treat "Prompting" as a math search problem. We want to find the string of text that maximizes the Score on a Golden Dataset.
Algorithm How It Works
Hill Climbing Start with a prompt. Make small random changes. If score improves, keep it. Repeat.
Gradient-Free Since we can't calculate gradients on discrete text tokens easily, we use black-box optimization methods.
Genetic Algorithms Create a "population" of 10 prompts. Breed the best ones (mix their words). Mutate (swap words). Repeat for 50 generations.

4Ô∏è‚É£ Reinforcement Learning (RL)

Core Idea: Use a Reward Model to train a "Prefix" (soft prompt) or guide the generation.

üöÄ Auto-Prompting Pipeline

1. Generator: Creates 10 candidate prompts.
2. Evaluator: Tests them on 50 questions.
3. Updater: Uses RL to update the Generator to produce better candidates next time.

5Ô∏è‚É£ Prompt Distillation

Core Idea: Taking a huge, complex prompt (Chain-of-Thought) and teaching a smaller model to do the same thing with a shorter prompt.
Teacher (GPT-4)
Prompt: "Think step by step..." (500 tokens).
Output: Correct Answer.
Student (Llama-3-8B)
Prompt: "Solve." (10 tokens).
Output: Correct Answer (Learned from Teacher).

6Ô∏è‚É£ DSPy: Programming, Not Prompting

Core Idea: Stanford's DSPy framework treats prompts like code modules that can be compiled. You define the logic, and the Optimizer writes the prompt for you.
üõ†Ô∏è How DSPy Works:
1. Signatures: Define what the task is (e.g., "question -> answer").
2. Modules: Choose a pattern (e.g., ChainOfThought, ReAct).
3. Optimizer (Teleprompter): Automatically searches for the best "few-shot" examples and instructions to maximize your metric.

‚ú® The Paradigm Shift

Instead of rewriting: "Try to be helpful and use bullet points..."
You just run: optimizer.compile(my_program, trainset=dataset)

7Ô∏è‚É£ Soft vs. Hard Prompting

Core Idea: Automation can happen at the Text level (words we can read) or the Vector level (numbers the AI reads).
Type Mechanism Pros/Cons
Hard Prompting Searching for actual words/tokens (e.g., Genetic algorithms). ‚úÖ Human-readable. ‚ùå Hard to optimize with math.
Soft Prompting Adding trainable vectors (numbers) to the start of the input. ‚úÖ Extremely efficient math optimization. ‚ùå Gibberish to humans.
